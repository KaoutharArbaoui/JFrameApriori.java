#################################       Read me         ########################################################
1- Faire un run au projet.

2- sélectionner un des fichiers qui existent dans le package dataset.

3- choisir le séparateur qui existe dans le fichier choisi, le support et la confiance.

4- Pour les boutons Les transactions, les articles uniques et Nbrs article... fonctionne bien 
mais pour les 2 boutons qui restent se bloque, pour qu'il travaille il faut mettre un run une deuxième
fois saisir les paramètres puis cliquer directement sur le bouton ensembles des éléments fréquents.

5- Aussi pour visualiser le contenu du bouton générer les règles d'association, il faut répéter l'étape 4
et cette fois cliquer directement sur le bouton générer les règles d'association.

6- Si vous voulez voir l’affichage dans la console, il suffit d’enlever le commentaire et faire un run à la
classe Main.java, pour voir par exemple l’affichage des transactions et les règles d’associations.
